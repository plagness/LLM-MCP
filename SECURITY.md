# Security Policy

## Reporting a vulnerability
Please report vulnerabilities privately via GitHub Security Advisories or direct contact with the repository owner.

Do not open public issues for security-sensitive reports.

## Scope
- Source code in this repository
- CI/CD and container definitions
- Documented runtime and integration paths

## Response targets
- Initial triage: up to 72 hours
- Fix or mitigation plan: as soon as reproducible details are confirmed
